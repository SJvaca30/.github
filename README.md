# SJvaca30 default community files

Default community health files for repositories owned by `SJvaca30`.

## Templates

- `bug_report.yml`: structured defect and regression intake
- `feature_request.yml`: feature proposal with acceptance criteria
- `engineering_task.yml`: implementation, maintenance, docs, test, and CI work
- `technical_decision.yml`: technical option comparison and decision record
- `PULL_REQUEST_TEMPLATE.md`: PR summary, verification, risk, rollback, and AI tracking

These templates avoid project-specific assumptions and keep fixed headings so people and AI assistants can trace issues, branches, commits, pull requests, verification evidence, and follow-up work consistently.
